# trackIt
Create a habit tracker

## Installation
- Clone or download the repo
- Do ```cd trackIT/api/``` then ```npm i```

## Usage
- Open terminal and navigate to the repo folder
- ```cd api``` then ```npm run dev``` to run development
- To test use ```npm test``` in api folder
